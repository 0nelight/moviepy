Decorators¶

These decorators are implemented to ease the writing of methods and effects in MoviePy

all decorators used in moviepy go there

moviepy.decorators.apply_to_audio(f)[source]¶

This decorator will apply the function f to the audio of the clip created with f

moviepy.decorators.apply_to_mask(f)[source]¶

This decorator will apply the same function f to the mask of the clip created with f

moviepy.decorators.audio_video_fx(f)[source]¶

This decorator tells that the function f (audioclip -> audioclip) can be also used on a video clip, at which case it returns a videoclip with modified audio.

moviepy.decorators.requires_duration(f)[source]¶

will raise an error if the clip has no duration.

moviepy.decorators.time_can_be_tuple(f)[source]¶

This decorator tells that the function f (audioclip -> audioclip) can be also used on a video clip, at which case it returns a videoclip with modified audio.
