AudioClip¶

class AudioClip.AudioClip[source]¶

Bases: Clip.Clip

Base class for audio clips. See SoundClip and CompositeSoundClip for usable classes.

Variables:
  • get_frame – A function t-> frame at time t. the frame does not mean much for a sound, it is just an integer.
  • nchannels – Number of channels (one or two for mono or stereo) of the sound.
  • nbits – Number of bits per frame (the more the better).
fadein(duration)[source]¶

Return a sound clip that is first mute, then the sound arrives progressively over duration seconds.

fadeout(duration)[source]¶

Return a sound clip where the sound fades out progressively over duration seconds at the end of the clip.

preview(fps=22050, sleep=True)[source]¶

Plays the sound clip with pygame. If sleep=True the program is freezed while the sound is playing (necessary when the sound is playing alone without video).

SoundClip¶

class AudioClip.SoundClip(snd, fps=None, bitrate=3000)[source]¶

Bases: AudioClip.AudioClip

An audio clip read from a sound file, or an array.

Parameters:
  • snd – Either a soundfile or an array representing a sound. If the soundfile is not a .wav, it will be converted to .wav first, using the fps and bitrate arguments.
  • fps – If snd is an array, this parameter is required.
  • bitrate – If snd is not a .wav, this prameter is required for the conversion.
 
>>> snd = SoundClip("song.wav")
>>> snd = SoundClip("song.mp3", fps = 44100, bitrate=3000)
>>> snd = SoundClip(mySoundArray,fps=44100) # from a numeric array

CompositeAudioClip¶

class AudioClip.CompositeAudioClip(clips)[source]¶

Bases: AudioClip.AudioClip

An audio clip made by putting together several audio clips.

Parameters:clips – a list of audio clips, which may start playing at different times or together. If all have their duration attribute set, the duration of the composite clip is computed automatically.
